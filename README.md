# Hue to Json
Speichert alle Parameter eines Hue-Systems in einer JSON-Datei. Die zu speichernden Parameter können dabei entsprechend angewählt werden, zur Verfügung stehen:

* Lights
* Groups
* Schedules
* Scenes
* Sensors
* Rules
* Configuration
* Capability
* ResourceLinks
